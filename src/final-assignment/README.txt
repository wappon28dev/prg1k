Cook Zip - C Impl.

概要:
このプログラムは，プロ演習Ⅰの演習問題や課題を提出する際の命名規則や ZIP ファイルの作成を自動化します.
ファイルの読み書き，学籍番号のバリデーション，ZIP ファイルの作成をすべて C 言語で行い，ユーザーにリッチな TUI を提供します.

機能:
名前の一括変更，ファイル圧縮，学籍番号のバリデーション，ファイルの接頭辞や接尾辞の推測/追加，ZIP の作成と Finder での表示

想い:
最終課題の前から，このような提出 ZIP 作成の自動化を既に TypeScript で実装しており，10 人弱の子に使ってもらっているので，仕様決めはスムーズに進みました.
`map` や `filter` などの便利関数が恋しかったですがが，Result 型を持ち込んだり，Doxygen を書いたりと技術的挑戦が多く，楽しかったです.
